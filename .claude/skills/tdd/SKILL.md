# 테스트 주도 자율 반복 스킬

테스트가 통과할 때까지 자율적으로 반복하는 TDD 워크플로우입니다.

## 핵심 원칙

**중간에 사용자에게 확인을 요청하지 않습니다. 모든 테스트가 통과할 때까지 혼자서 계속 반복합니다.**

## 실행 단계

1. **실패 테스트 먼저 작성**
   - 예상 동작을 정의하는 테스트 파일 생성
   - Remotion: `*.test.tsx` (Jest/Vitest)
   - Python: `test_*.py` (pytest)

2. **테스트 실행**
   - Remotion: `cd remotion && npm test`
   - Python: `cd python && pytest`

3. **최소 코드 구현**
   - 테스트를 통과할 최소한의 코드만 작성

4. **테스트 재실행**

5. **실패 시 자동 수정**
   - 오류 출력 분석
   - 사용자에게 묻지 않고 직접 수정
   - TodoWrite로 남은 실패 테스트 추적

6. **4-5단계 반복**
   - 모든 테스트가 통과할 때까지 계속

7. **완료 보고**
   - 모든 테스트 통과 후에만 결과 보고
   - 테스트 출력 포함
   - 구현된 기능 요약

## 주의사항

- 반복 중 확인 요청 금지
- 실패 원인을 스스로 분석하고 수정
- 막히면 다른 접근 방식 시도
- 최대 10회 반복 후에도 실패하면 현재 상태 보고
